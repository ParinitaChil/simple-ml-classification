# Simple ML Classification - Iris Dataset

## Overview
This project demonstrates a **beginner-friendly machine learning workflow** using the classic Iris dataset.  
It covers data loading, exploration, model training, evaluation, and visualization.

## Dataset
- **Iris dataset** (built-in from scikit-learn)
- Contains 150 samples of iris flowers
- Features: sepal length, sepal width, petal length, petal width
- Target: species (0 = setosa, 1 = versicolor, 2 = virginica)

## Tools / Libraries
- Python
- Pandas
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

## Steps Performed
1. Load the Iris dataset into a Pandas DataFrame  
2. Explore the data (head, info, descriptive statistics)  
3. Visualize features (scatter plot of sepal length vs width)  
4. Train/Test split for modeling  
5. Train a **Logistic Regression** classifier  
6. Evaluate model accuracy on the test set

## Key Findings
- Logistic Regression achieves high accuracy on the Iris dataset  
- Visualization helps understand feature separation between classes  
- Demonstrates a complete ML workflow for beginners

## How to Run
1. Open `simple_classification.ipynb` in **Jupyter Notebook** or **Google Colab**  
2. Run all cells step-by-step to see data exploration, modeling, and results
